# Sentinel Monitor Logo设计规格

## 设计概览
- **品牌名称**：哨兵监控 (Sentinel Monitor)
- **设计风格**：抽象图标型
- **颜色方案**：活力监控橙
- **核心理念**：安全守护 + 实时监控 + 数据洞察

## 设计概念

### 核心元素融合
将以下三个概念融合为简洁的抽象图标：
1. **哨兵/守护** - 盾牌、城堡、守护者
2. **监控/观测** - 眼睛、雷达、扫描
3. **数据/连接** - 波形、节点、网络

### 最终概念：守护之眼
一个抽象的"眼睛"被"盾牌"保护，同时融入"雷达波"元素，体现全天候安全监控。

## 详细设计规格

### 1. 基础形状
```
[盾牌外框] + [眼睛瞳孔] + [雷达波纹]
```

**结构描述：**
- **外层**：简约盾牌形状（圆角六边形）
- **中层**：抽象眼睛形状（椭圆形）
- **内层**：雷达波纹（同心圆或扇形波纹）
- **整体**：中心对称，平衡稳定

### 2. 颜色方案

#### 主色系（活力监控橙）
- **主色**：`#FF6D00` - 活力橙（用于核心元素）
- **辅助色**：`#2D5BFF` - 科技蓝（用于细节和强调）
- **背景色**：透明 或 `#FEFCE8`（浅黄，可选）
- **文字色**：`#1E293B`（深蓝灰，用于文字搭配）

#### 使用规则
- **图标主体**：主色 `#FF6D00`
- **细节强调**：辅助色 `#2D5BFF`（占比不超过30%）
- **背景**：透明或浅色背景
- **单色版本**：`#1E293B`（深蓝灰）

### 3. 尺寸规格

#### 主要版本
| 用途 | 尺寸 | 文件格式 | 说明 |
|------|------|---------|------|
| **主Logo** | 1024×1024px | PNG, SVG | 完整尺寸，透明背景 |
| **网站Favicon** | 32×32px, 16×16px | ICO, PNG | 浏览器标签图标 |
| **网页头部** | 256×256px | PNG | 导航栏Logo |
| **移动端** | 128×128px | PNG | 移动设备显示 |
| **打印文档** | 300×300px @300DPI | PDF, EPS | 印刷质量 |

#### 变体要求
- 所有尺寸保持相同比例和视觉权重
- 小尺寸简化细节，保持可识别性
- 提供圆形、方形、横版三种布局变体

### 5. 设计要点

#### 抽象化处理
1. **盾牌**：不要具象的盾牌细节，使用简约的六边形或菱形变体
2. **眼睛**：不要真实的眼球，使用椭圆形或杏仁形抽象
3. **雷达波**：使用虚线圆环或扇形扫描线

#### 视觉平衡
- **重心**：中心偏上，体现"守护"的稳重感
- **比例**：盾牌:眼睛:雷达 = 3:2:1（面积比）
- **负空间**：合理利用负空间，保持呼吸感

#### 细节处理
- **圆角**：所有尖角使用8-12px圆角
- **渐变**：主色可使用轻微渐变增加立体感（`#FF6D00` → `#FF8C00`）
- **光影**：如需立体感，添加轻微内阴影（`rgba(0,0,0,0.1)`）

### 6. 使用规范

#### 安全区域
- Logo周围保持最小空白区域（等于Logo高度的20%）
- 避免与其他图形元素重叠

#### 背景适用
- **浅色背景**：使用完整彩色版本
- **深色背景**：使用反白版本（主色改为白色，辅助色改为浅蓝）
- **单色背景**：使用单色版本

#### 禁用行为
- ❌ 不要拉伸变形
- ❌ 不要旋转角度（除特殊设计需求）
- ❌ 不要更改颜色组合
- ❌ 不要添加额外效果（阴影、发光等）

## 快速制作指南

### 使用在线工具（推荐Canva）
1. 访问 [canva.com](https://www.canva.com/)
2. 搜索"盾牌"、"眼睛"图标素材
3. 组合调整，使用指定颜色
4. 导出PNG和SVG格式

### 手动设计要点
1. **先画盾牌**：使用圆角六边形工具
2. **添加眼睛**：内部放置椭圆形
3. **绘制雷达**：添加虚线圆环
4. **调整颜色**：应用指定色值
5. **导出文件**：按规格导出多尺寸

### 设计调整建议
- 如果盾牌+眼睛+雷达的组合太复杂，可简化为**盾牌+雷达波**
- 或者简化为**眼睛+保护圈**的抽象形式
- 保持核心：**橙色主色 + 抽象几何 + 守护监控寓意**
