# 品牌命名与Logo设计提案

## 项目背景
基于开源的Pika监控系统，定制为自用版本。需要新的品牌名称和Logo设计，体现监控、安全、实时数据等核心功能。

## 设计原则
1. **简洁易记** - 名称简短，便于记忆和传播
2. **技术感强** - 体现监控、数据、安全等特性
3. **国际化** - 中英文名称协调，适合技术产品
4. **视觉识别** - Logo简洁、可缩放、有辨识度

## Logo设计提案

### 设计方向
基于现有Pika Logo（1024x1024像素PNG），提出以下设计方向：

#### 方向一：抽象图标型
- **概念**：使用抽象几何图形组合，体现数据流、监控、连接等概念
- **元素**：眼睛（监控）、雷达波（探测）、网络节点（连接）、盾牌（安全）
- **颜色**：科技蓝 (#2D5BFF) + 安全绿 (#00C853) 或 监控橙 (#FF6D00)
- **示例**：眼睛与雷达波结合的抽象图形

#### 方向二：字母组合型
- **概念**：使用品牌首字母设计简约图标
- **元素**：如"S"（哨兵）、"O"（观测者）等字母的变形设计
- **颜色**：单色或双色渐变，突出专业感
- **示例**：S字母融入眼睛或雷达元素

#### 方向三：具象符号型
- **概念**：使用具象但简化的图标，如眼睛、盾牌、雷达等
- **元素**：简化版的眼睛（监控）、盾牌（安全）、塔楼（哨兵）
- **颜色**：强调色+中性色搭配
- **示例**：简约的哨兵塔楼图标

### 颜色方案推荐

#### 方案A：专业科技蓝
- **主色调**：#2D5BFF（科技蓝）
- **辅助色**：#00C853（安全绿）
- **背景色**：#F8FAFC（浅灰白）
- **文字色**：#1E293B（深蓝灰）
- **特点**：专业、可靠、技术感强

#### 方案B：安全守护绿
- **主色调**：#00C853（安全绿）
- **辅助色**：#2D5BFF（科技蓝）
- **背景色**：#FFFFFF（纯白）
- **文字色**：#334155（深灰）
- **特点**：安全、环保、清新

#### 方案C：活力监控橙
- **主色调**：#FF6D00（监控橙）
- **辅助色**：#2D5BFF（科技蓝）
- **背景色**：#FEFCE8（浅黄）
- **文字色**：#1E293B（深蓝灰）
- **特点**：醒目、活力、易识别

### Logo技术规格
- **格式**：PNG（支持透明背景）
- **尺寸**：1024×1024像素（主文件）
- **变体**：不同尺寸适配（256×256, 128×128, 64×64, 32×32）
- **颜色模式**：RGB（屏幕显示）+ 单色（打印/灰度）
- **文件大小**：<100KB（优化后）

## 实施建议

### 命名实施步骤
1. **确认名称**：从提案中选择或调整
2. **域名检查**：检查相关域名可用性
3. **一致性验证**：确保中英文名称协调
4. **商标查询**：初步检查商标注册情况

### Logo制作流程
1. **概念设计**：基于选择的方向进行草图设计
2. **数字制作**：使用矢量工具（AI/Sketch/Figma）制作
3. **多尺寸适配**：生成不同尺寸版本
4. **格式导出**：导出PNG、SVG、ICO等格式
5. **应用测试**：在Web界面测试显示效果

### 现有系统适配
1. **系统名称配置**：在`internal/service/property_service.go`中修改默认名称
2. **Logo文件替换**：替换`web/public/logo.png`文件
3. **界面文字更新**：更新所有包含"Pika"的界面文字
4. **文档更新**：更新README.md等文档中的品牌信息
